Lightweight internal linking suggestions inside the WordPress block editor.

== Description ==
AP Internal Linking Helper adds a fast, distraction-free sidebar to the block editor that suggests internal links while you write.

Perfect for bloggers, niche sites, and small businesses who want better internal link coverage without complex SEO suites.

Features:
* Editor sidebar with internal link suggestions
* Shows recent and related items
* Copy link and open actions
* Simple, lightweight, no setup

== How It Works ==
Open the editor, enable the "AP Link Helper" panel, and use the suggestions to add internal links quickly.
