A lightweight internal linking helper that suggests relevant internal links inside the block editor.

== Description ==
AP Internal Linking Helper provides simple, safe internal link suggestions for your posts and pages.

== Installation ==
1. Upload the plugin to your /wp-content/plugins/ directory.
2. Activate through the 'Plugins' menu in WordPress.

== Frequently Asked Questions ==
= Does this work with the block editor? =
Yes.
